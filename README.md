# Weather-App-Design-Time_Pass-
Just Designing Weather App Design in Free Time

## Overview

This repository contains the design and styling for a weather app. It provides a visually appealing and user-friendly interface to display current weather information and a weekly forecast.

## Features

- Current weather information including temperature and description.
- Weekly forecast with day symbols, names, and temperatures.
- Additional details such as precipitation, wind speed, and humidity.
- Change location button for user flexibility.
